# 🎥 YouTube Clone

A responsive YouTube homepage clone built using **HTML, CSS, and JavaScript**. This project recreates the core YouTube interface to practice front-end development, responsive layouts, and DOM manipulation.

## 🚀 Features

- Responsive YouTube-inspired layout
- Fixed header and sidebar
- Search bar with tooltips
- Responsive video grid using CSS Grid
- Clickable video thumbnails
- Clean and organized file structure
- Modern UI built with Flexbox and Grid

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure
